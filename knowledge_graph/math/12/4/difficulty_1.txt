# HƯỚNG DẪN SINH ĐỀ MỨC NHẬN BIẾT
## CHỦ ĐỀ: NGUYÊN HÀM - TÍCH PHÂN (LỚP 12)

## I. NGUYÊN HÀM

### 1. Định nghĩa và tính chất cơ bản

**Dạng câu hỏi:**
- **Cho biết** định nghĩa nguyên hàm của hàm số f(x)
- **Nhận biết** công thức nguyên hàm cơ bản
- **Xác định** tính chất của nguyên hàm

**Ví dụ:**
- Cho biết F(x) là một nguyên hàm của f(x) trên khoảng I khi nào?
- Nếu F(x) là nguyên hàm của f(x) thì họ nguyên hàm của f(x) là gì?

### 2. Bảng nguyên hàm cơ bản

**Dạng câu hỏi:**
- **Tính** nguyên hàm của các hàm cơ bản: x^n, e^x, 1/x, sin x, cos x
- **Tìm** nguyên hàm áp dụng trực tiếp công thức trong bảng

**Ví dụ mẫu:**
```
Câu 1: Tính ∫x³dx
A. x⁴ + C    B. x⁴/4 + C    C. 3x² + C    D. x⁴/3 + C

Câu 2: Nguyên hàm của hàm số f(x) = cos x là:
A. -sin x + C    B. sin x + C    C. -cos x + C    D. tan x + C

Câu 3: Cho biết ∫(1/x)dx bằng:
A. ln|x| + C    B. -1/x² + C    C. x + C    D. 1/x² + C
```

### 3. Tính chất tuyến tính

**Dạng câu hỏi:**
- **Tính** nguyên hàm của tổng, hiệu các hàm
- **Xác định** nguyên hàm có hệ số

**Ví dụ mẫu:**
```
Câu 4: Tính ∫(2x + 3)dx
A. 2x² + 3x + C    B. x² + 3x + C    C. 2x + C    D. x² + 6x + C

Câu 5: Nguyên hàm của f(x) = 3x² - 2x + 1 là:
A. x³ - x² + x + C    B. 3x³ - 2x² + x + C    
C. x³ - x² + x + C    D. 6x - 2 + C
```

## II. TÍCH PHÂN

### 1. Định nghĩa tích phân xác định

**Dạng câu hỏi:**
- **Cho biết** định nghĩa tích phân xác định
- **Nhận biết** ký hiệu và ý nghĩa hình học
- **Xác định** các thành phần trong ký hiệu tích phân

**Ví dụ mẫu:**
```
Câu 6: Trong ký hiệu ∫[a to b]f(x)dx, a và b được gọi là gì?
A. Cận trên và cận dưới    B. Cận dưới và cận trên    
C. Hàm số và biến số       D. Nguyên hàm và đạo hàm
```

### 2. Công thức Newton-Leibniz

**Dạng câu hỏi:**
- **Tính** tích phân xác định áp dụng trực tiếp công thức
- **Xác định** giá trị tích phân đơn giản

**Ví dụ mẫu:**
```
Câu 7: Tính ∫[0 to 1]x²dx
A. 1/3    B. 1    C. 2/3    D. 1/2

Câu 8: Cho biết ∫[0 to π/2]cos x dx bằng:
A. 0    B. 1    C. -1    D. π/2

Câu 9: Tính ∫[1 to 2](2x + 1)dx
A. 6    B. 5    C. 4    D. 7
```

### 3. Tính chất của tích phân

**Dạng câu hỏi:**
- **Nhận biết** tính chất tuyến tính
- **Xác định** tính chất đổi cận
- **Cho biết** tính chất cộng cận

**Ví dụ mẫu:**
```
Câu 10: Cho biết ∫[a to a]f(x)dx bằng:
A. 1    B. 0    C. a    D. f(a)

Câu 11: Nếu ∫[a to b]f(x)dx = 5 thì ∫[b to a]f(x)dx bằng:
A. 5    B. -5    C. 0    D. 10

Câu 12: Cho ∫[0 to 2]f(x)dx = 3 và ∫[2 to 5]f(x)dx = 7. 
Tính ∫[0 to 5]f(x)dx
A. 4    B. 10    C. -4    D. 21
```

## III. ỨNG DỤNG CƠ BẢN

### 1. Diện tích hình phẳng đơn giản

**Dạng câu hỏi:**
- **Nhận biết** công thức tính diện tích
- **Xác định** cận tích phân cho sẵn

**Ví dụ mẫu:**
```
Câu 13: Diện tích hình phẳng giới hạn bởi y = x², trục Ox, 
x = 0, x = 1 được tính bằng công thức:
A. ∫[0 to 1]x²dx    B. ∫[0 to 1]x dx    
C. ∫[0 to 1]2x dx   D. ∫[0 to 1]x³dx
```

## IV. HƯỚNG DẪN SOẠN ĐỀ

### Cấu trúc đề thi mức Nhận biết:
- **40% câu hỏi về nguyên hàm cơ bản**
- **40% câu hỏi về tích phân xác định**  
- **20% câu hỏi về ứng dụng đơn giản**

### Lưu ý khi soạn đề:
1. Sử dụng các hàm số đơn giản: đa thức bậc thấp, lượng giác cơ bản
2. Cận tích phân là các số đẹp, dễ tính
3. Đáp án phải rõ ràng, không gây nhầm lẫn
4. Tránh các phép biến đổi phức tạp
5. Ưu tiên công thức có trong bảng nguyên hàm cơ bản
