# TÓM TẮT CHƯƠNG ĐẠO HÀM - TOÁN 11

## I. KHÁI NIỆM ĐẠO HÀM

### 1. Bài toán dẫn đến khái niệm đạo hàm

**Bài toán 1: Vận tốc tức thời**
Một vật chuyển động theo phương trình s = s(t). Tại thời điểm t₀, vận tốc tức thời của vật là:
```
v(t₀) = lim[Δt→0] [s(t₀ + Δt) - s(t₀)]/Δt
```

**Bài toán 2: Tiếp tuyến của đường cong**
Hệ số góc của tiếp tuyến tại điểm M₀(x₀; f(x₀)) trên đồ thị hàm số y = f(x) là:
```
k = lim[Δx→0] [f(x₀ + Δx) - f(x₀)]/Δx
```

### 2. Định nghĩa đạo hàm tại một điểm

Cho hàm số y = f(x) xác định trên khoảng (a; b) chứa điểm x₀.

**Đạo hàm của hàm số f(x) tại điểm x₀** là giới hạn (nếu tồn tại):
```
f'(x₀) = lim[h→0] [f(x₀ + h) - f(x₀)]/h
```

**Các ký hiệu khác:** y'(x₀), df/dx|_{x=x₀}

**Chú ý:** 
- Nếu giới hạn trên tồn tại và hữu hạn thì ta nói hàm số f(x) có đạo hàm (hay khả vi) tại x₀
- Nếu f(x) có đạo hàm tại x₀ thì f(x) liên tục tại x₀

### 3. Đạo hàm một bên

**Đạo hàm phải:** f'₊(x₀) = lim[h→0⁺] [f(x₀ + h) - f(x₀)]/h

**Đạo hàm trái:** f'₋(x₀) = lim[h→0⁻] [f(x₀ + h) - f(x₀)]/h

**Định lý:** f'(x₀) tồn tại ⟺ f'₊(x₀) = f'₋(x₀) và cả hai đều hữu hạn

### 4. Hàm đạo hàm

Nếu hàm số f(x) có đạo hàm tại mọi điểm x thuộc khoảng (a; b), ta có **hàm đạo hàm**:
```
f'(x) = lim[h→0] [f(x + h) - f(x)]/h
```

## II. QUY TẮC TÍNH ĐẠO HÀM

### 1. Đạo hàm của các hàm số cơ bản

| Hàm số | Đạo hàm | Điều kiện |
|--------|---------|-----------|
| f(x) = c | f'(x) = 0 | c ∈ ℝ |
| f(x) = x | f'(x) = 1 | |
| f(x) = xⁿ | f'(x) = nxⁿ⁻¹ | n ∈ ℝ, x > 0 |
| f(x) = √x | f'(x) = 1/(2√x) | x > 0 |
| f(x) = 1/x | f'(x) = -1/x² | x ≠ 0 |

### 2. Quy tắc tính đạo hàm

**Quy tắc 1 (Tính chất tuyến tính):**
```
[u(x) ± v(x)]' = u'(x) ± v'(x)
[ku(x)]' = ku'(x) (k là hằng số)
```

**Quy tắc 2 (Đạo hàm tích):**
```
[u(x).v(x)]' = u'(x).v(x) + u(x).v'(x)
```

**Quy tắc 3 (Đạo hàm thương):**
```
[u(x)/v(x)]' = [u'(x).v(x) - u(x).v'(x)]/[v(x)]² (v(x) ≠ 0)
```

**Quy tắc 4 (Đạo hàm hàm hợp):**
Nếu y = f(u), u = g(x) thì:
```
y'ₓ = y'ᵤ . u'ₓ
```
hay [f(g(x))]' = f'(g(x)).g'(x)

### 3. Đạo hàm của hàm số lượng giác

| Hàm số | Đạo hàm | Điều kiện |
|--------|---------|-----------|
| sin x | cos x | |
| cos x | -sin x | |
| tan x | 1/cos²x = 1 + tan²x | x ≠ π/2 + kπ |
| cot x | -1/sin²x = -(1 + cot²x) | x ≠ kπ |

### 4. Ví dụ minh họa

**Ví dụ 1:** Tính đạo hàm của f(x) = 3x⁴ - 2x² + 5x - 1

*Giải:*
f'(x) = (3x⁴)' - (2x²)' + (5x)' - (1)'
= 3.4x³ - 2.2x + 5 - 0
= 12x³ - 4x + 5

**Ví dụ 2:** Tính đạo hàm của g(x) = (2x + 1)(x² - 3)

*Giải:*
Cách 1: g'(x) = (2x + 1)'(x² - 3) + (2x + 1)(x² - 3)'
= 2(x² - 3) + (2x + 1).2x
= 2x² - 6 + 4x² + 2x
= 6x² + 2x - 6

**Ví dụ 3:** Tính đạo hàm của h(x) = sin(2x + 1)

*Giải:*
Đặt u = 2x + 1, ta có h(x) = sin u
h'(x) = (sin u)' . u' = cos u . 2 = 2cos(2x + 1)

## III. Ý NGHĨA HÌNH HỌC VÀ VẬT LÝ CỦA ĐẠO HÀM

### 1. Ý nghĩa hình học

**Định lý:** Đạo hàm f'(x₀) chính là hệ số góc của tiếp tuyến với đồ thị hàm số y = f(x) tại điểm M(x₀; f(x₀)).

**Phương trình tiếp tuyến:**
```
y = f'(x₀)(x - x₀) + f(x₀)
```

**Phương trình pháp tuyến:**
```
y = -1/f'(x₀)(x - x₀) + f(x₀) (f'(x₀) ≠ 0)
```

### 2. Ý nghĩa vật lý

- **Vận tốc tức thời:** Nếu s = s(t) là phương trình chuyển động, thì v(t) = s'(t)
- **Gia tốc tức thời:** a(t) = v'(t) = s''(t)

### 3. Ví dụ ứng dụng

**Ví dụ 1:** Viết phương trình tiếp tuyến của đồ thị hàm số y = x³ - 3x + 1 tại điểm có hoành độ x₀ = 1.

*Giải:*
- Tại x₀ = 1: y₀ = 1³ - 3.1 + 1 = -1
- y' = 3x² - 3, nên y'(1) = 3.1² - 3 = 0
- Phương trình tiếp tuyến: y = 0(x - 1) + (-1) = -1

**Ví dụ 2:** Một vật chuyển động theo phương trình s = t³ - 6t² + 9t (m). Tính vận tốc và gia tốc tại t = 2s.

*Giải:*
- v(t) = s'(t) = 3t² - 12t + 9
- a(t) = v'(t) = 6t - 12
- Tại t = 2: v(2) = 3.4 - 12.2 + 9 = -3 (m/s)
- a(2) = 6.2 - 12 = 0 (m/s²)
